Python For Data Science

The course you need to learn python programming language and its application in Data Science.
Details Syllabus

Requirements

  • Have a computer with Internet connection.
  • Be ready to learn an insane amount of awesome stuff.
  • Prepare to analyze real data.

Description

Python is a dynamic, powerful, and easy to learn programming language. This course introduces you to python with a focus on how you can start applying it in your data science projects.
Python provides a wide range of inbuilt functionality that you can use in your own projects. The functionality is grouped into libraries. Some of the libraries covered in this course are: NumPy, Pandas, Seaborn, and Matplotlib. We will use each library in a mini-project.

Prerequisites

  • Critical Thinking.
  • Minimum of Bachelor's Degree/Higher Diploma in Sciences, Engineering and Social Sciences.

What you will learn

  • Introduction to Programming?
  • Introduction to Python
  • Introduction to the Cloud IDE for Python
  • Variables in Python
  • Functions in Python
  • Capstone 1: Python program that makes use of functions
  • Career Coaching Check-In
  • Working With Data Files
  • Capstone 2: Python program that prints graphs and charts
  • Career Coaching Check-In
